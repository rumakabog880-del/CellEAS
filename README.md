# CellEAS

Local Windows host tool for testing Android Emulator emergency Cell Broadcast UI.



# FOR LOCAL TESTING ONLY. THE AUTHOR IS NOT RESPONSIBLE FOR ANY USE.



## What It Does

`CellBroadcastLab.exe` is only a host-side controller. It does not install an Android app that draws an alert.

The send path is:

1. Host pushes `cellbroadcast-injector.jar` to `/data/local/tmp/` inside a selected `emulator-*` device.
2. Host runs it with Android `app_process`.
3. The injector creates a real `android.telephony.SmsCbMessage` parcelable with CMAS metadata.
4. The injector sends Android's protected emergency Cell Broadcast intent:
   `android.provider.action.SMS_EMERGENCY_CB_RECEIVED`.
5. The platform CellBroadcastReceiver package receives the message and displays the system alert dialog.

The tool refuses non-`emulator-*` serials in the GUI.

## First Run

1. Run `Run-CellBroadcastLab.bat` or `CellBroadcastLab.exe`.
2. Click `Install / build SDK`.
   This downloads a local portable JDK, Android command-line tools, platform-tools, emulator, build-tools, and an Android x86_64 emulator system image into the workspace `work/` folder.
3. Wait for the installer window to finish.
4. Click `Start emulator`.
5. After Android boots, click `Refresh devices`.
6. Click `Check emulator`.
7. Edit text/type/language if desired.
8. Click `Send system emergency alert`.

## Files

- `CellBroadcastLab.exe`: Windows GUI.
- `Run-CellBroadcastLab.bat`: launcher.
- `Install-AndroidSdk.ps1`: local SDK/JDK/AVD bootstrap and injector build.
- `cellbroadcast-injector.jar`: generated by the installer.
- `injector-src/`: source for the in-emulator injector.

## Notes

- Use a root-friendly emulator image, not a Play Store image, so `adb root` is available.
- The alert may be filtered if the emulator image lacks a CellBroadcastReceiver package or carrier config disables that alert class.
- The GUI's `Receiver package` field defaults to `auto`; the injector resolves packages containing `cellbroadcast`.
- This is local emulator testing only. It does not use or contact real cellular networks.
